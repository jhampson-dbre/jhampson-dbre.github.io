Home Assistant Collection for Ansible

Install, configure, and secure Home Assistant on a Raspberry Pi running Debian.

Technology
Home Assistant Collection includes 6 Ansible roles to automate the installation of Home Assistant. The collection enhances standard Home Assistant install by setting up automatic Debian security updates using unattended-upgrades and configuring fail2ban to report on unauthorized remote login attempts.

A GitHub Actions workflow lints and tests new releases of Home Assistant Collection before publishing to Ansible Galaxy.

Nginx Reverse Proxy with Web Application Firewall Add-on for Home Assistant

Securely expose your Home Assistant installation to the internet with this add-on.

Technology
Nginx Reverse Proxy with Web Application Firewall Add-on is a fork of the standard Nginx Reverse Proxy add-on, but with an additional layer of security from modsecurity web application firewall pre-loaded with the OWASP core rule set.

The add-on runs inside of a Docker container. New releases of the add-on are published to Docker Hub using GitHub Actions.
